### fzf

執行下面指令，提供fzf列表選項安裝

``` sh
leftwmrc-theme-get fzf
```

### all

執行下面指令，安裝以下所有的「theme」

``` sh
leftwmrc-theme-get all
```

### dracula_rounded

執行下面指令，單一安裝「theme / [dracula_rounded](https://github.com/AethanFoot/leftwm-theme-dracula-rounded/)」

``` sh
leftwmrc-theme-get dracula_rounded
```
