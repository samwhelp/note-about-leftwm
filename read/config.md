## 如何安裝

執行下面指令

``` sh
curl -fLo ~/.config/leftwm/config.toml --create-dirs \
	'https://raw.githubusercontent.com/samwhelp/note-about-leftwm/gh-pages/_demo/config/main/config/leftwm/config.toml'
```
