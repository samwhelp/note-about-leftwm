# 其他視窗操作


## 關閉視窗

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + q` | 將目前聚焦的視窗關閉 |


## 全螢幕

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + f` | 將目前聚焦的視窗，切換成全螢幕 |


## 交換到主要視窗位置

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + b` | 將目前聚焦的視窗，交換到主要視窗位置 |


## 從浮動模式和切換到平鋪模式

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + v` | 將目前聚焦的視窗，從浮動模式和切換到平鋪模式 |
