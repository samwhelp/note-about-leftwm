# 其他視窗操作

## 按鍵功能導向

| 按鍵 | 功能導向 |
| --- | --- |
| [Win](#win) | Window |


## 關閉視窗

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + q` | 將目前聚焦的視窗關閉 |


## 全螢幕

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + f` | 將目前聚焦的視窗，切換成全螢幕 |


## 放到主要位置

| 按鍵組合 | 功能 |
| --- | --- |
| `Win + w` | 將目前聚焦的視窗，放到主要Layout位置 |
