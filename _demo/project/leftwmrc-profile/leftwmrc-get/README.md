
# leftwmrc-get

## Install

Run

``` sh
mkdir -p "$HOME/.local/bin"

wget 'https://raw.githubusercontent.com/samwhelp/note-about-leftwm/gh-pages/_demo/project/leftwmrc-profile/leftwmrc-get/leftwmrc-get' -O "$HOME/.local/bin/leftwmrc-get"

chmod 755 "$HOME/.local/bin/leftwmrc-get"
```

Or Run

``` sh
curl -fsSL 'https://raw.githubusercontent.com/samwhelp/note-about-leftwm/gh-pages/_demo/project/leftwmrc-profile/leftwmrc-get/remote-install.sh' | bash
```

Or Run

``` sh
wget 'https://raw.githubusercontent.com/samwhelp/note-about-leftwm/gh-pages/_demo/project/leftwmrc-profile/leftwmrc-get/remote-install.sh' -q -O - | bash
```


## Usage


### Help

``` sh
$ leftwmrc-get
```

or

``` sh
$ leftwmrc-get help
```


### Debug

``` sh
$ export DEBUG_LEFTWMRC_GET=true
```
